🐾 Sistema de Dicas - Pet Shop Baronesa

Sistema dinâmico para gerenciar dicas do pet shop. Use este guia para testar todas as funcionalidades.

✅ Sistema Configurado para GitHub Pages!

O sistema foi otimizado para funcionar perfeitamente em hospedagens como GitHub Pages, Netlify e Vercel.

  • 🔗 Caminhos automáticos resolvidos
  • 📱 Compatível com mobile
  • ⚡ Performance otimizada
  • 🔄 Atualizações em tempo real

🚀 Deploy no GitHub Pages

Para hospedar no GitHub Pages:

  1. Faça upload dos arquivos para um repositório GitHub
  2. Vá em Settings → Pages
  3. Selecione "main branch" e "/ (root)"
  4. Seu site estará em: https://seu-usuario.github.io/nome-repositorio

  • ✅ Integração com Firebase/Firestore (tabela "Dicas")
  • ✅ CRUD completo de dicas no painel admin

1. Acesse o Painel Administrativo

Abra o painel admin para gerenciar dicas:

🛠️ Abrir Admin

2. Navegue para a Aba "Dicas"

No painel admin, clique na aba "Dicas" para acessar o gerenciamento de dicas.

3. Adicione uma Nova Dica

Clique em "Nova Dica" e preencha os campos:

  • Título: Nome da dica
  • Categoria: Cachorros, Gatos, Pássaros ou Geral
  • Status: Rascunho ou Publicada
  • Resumo: Texto curto para o card
  • Conteúdo: Texto completo da dica
  • Imagem: URL da imagem (opcional)
  • Tags: Palavras-chave (opcional)

4. Visualize as Dicas Públicas

Acesse a página pública para ver as dicas publicadas:

👁️ Ver Dicas Públicas

5. Teste os Filtros

Na página pública, teste:

  • Filtros por categoria (Cachorros, Gatos, etc.)
  • Busca por texto
  • Navegação entre filtros

⚠️ Importante:

Apenas dicas com status "Publicada" aparecerão na página pública. Dicas "Rascunho" ficam visíveis apenas no admin.

🗄️ Estrutura do Banco de Dados:

Coleção: Dicas

Campos principais:

  • conteudo: Texto da dica
  • dataDePublicacao: Data de publicação
  • fileURL: URL da imagem
  • title: Título da dica
  • category: Categoria
  • status: published/draft
  • resumo: Resumo para card
  • tags: Array de tags
